# Prompt Manager

#### A MERN project to manage and optimize your prompt engineering workflow.

## Description

Prompt Manager: This project allows users to manage and optimize their prompt engineering workflow. It provides a user-friendly interface to create, edit, and delete prompts, as well as a history of previous prompts. Additionally, it uses Mistral model to optimize the prompts.

## Features

- Create, edit, and delete prompts.
- Save and load previous prompts.
- Receive feedback for optimizing your prompts.
